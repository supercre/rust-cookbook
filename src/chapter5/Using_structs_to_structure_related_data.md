# 구조체로 관련된 데이터를 구조화하기

_구조체(struct)_ 또는 _구조(structure)_는 의미 있는 그룹을 구성하는 여러 관련 값들을 함께 패키징하고 이름을 지정할 수 있는 사용자 정의 데이터 타입입니다. 객체 지향 언어에 익숙하다면, _구조체(struct)_는 객체의 데이터 속성과 같습니다. 이 장에서는 이미 알고 있는 내용을 바탕으로 튜플(tuple)과 구조체(struct)를 비교하고 대조하여, 구조체가 데이터를 그룹화하는 더 나은 방법이 되는 경우를 보여드리겠습니다.

구조체를 정의하고 인스턴스화하는 방법을 보여드리겠습니다. 구조체 타입과 관련된 동작을 지정하기 위해 연관 함수(associated function), 특히 _메서드(method)_라고 불리는 종류의 연관 함수를 정의하는 방법에 대해 논의할 것입니다. 구조체와 열거형(enum)(6장에서 논의)은 프로그램의 도메인에서 새로운 타입을 생성하기 위한 기본 구성 요소로, Rust의 컴파일 타임 타입 검사를 최대한 활용할 수 있게 해줍니다.
